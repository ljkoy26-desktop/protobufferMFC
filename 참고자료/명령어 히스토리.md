# filter_data.proto 파일 생성 

syntax = "proto3";

package my_msg;

// 필터 상세 내용 (Include/Exclude 배열)
message FilterItem {
    repeated string include_filters = 1;  // repeated는 std::vector와 같습니다.
    repeated string exclude_filters = 2;
}

// 전체 설정 (Key: string, Value: FilterItem 구조체)
message FilterConfig {
    map<string, FilterItem> filter_map = 1; // std::map과 같습니다.
}

Step 3. Visual Studio 2019 프로젝트에 추가 및 설정

생성된 filter_data.pb.h와 filter_data.pb.cc를 MFC 프로젝트 폴더로 복사합니다.
