예시) 

📁 내 프로젝트 폴더 ( .vcxproj 파일이 있는 곳 )
└── 📁 External
    └── 📁 protobuf
        ├── 📁 include   <-- [원본의 include 폴더 통째로 복사]
        ├── 📁 lib       <-- [원본의 lib 폴더 안의 Release용 .lib 파일들 복사]
        │   └── 📁 debug <-- [원본의 debug\lib 폴더 안의 Debug용 .lib 파일들 복사]
        └── 📁 bin       <-- [원본의 bin 폴더 안의 Release용 .dll 파일들 복사 (존재할 경우)]
            └── 📁 debug <-- [원본의 debug\bin 폴더 안의 Debug용 .dll 파일들 복사 (존재할 경우)]



            
좌측 메뉴에서 [C/C++] -> [일반]으로 이동합니다.
[추가 포함 디렉터리] 항목을 클릭하고 아래의 경로를 입력합니다.

# $(ProjectDir)External\protobuf\include


../include

[링커] -> [일반] -> [추가 라이브러리 디렉터리] 항목에 디버그 경로를 입력합니다.

# $(ProjectDir)External\protobuf\lib\debug
# $(ProjectDir)External\protobuf\lib


../lib64/protobuf/debug;
../lib64/protobuf/release;

[링커] -> [입력] -> [추가 종속성] 항목 맨 앞

libprotobufd.lib;
libprotobuf.lib;


libprotobufd.lib;abseil_dll.lib;utf8_range.lib;utf8_validity.lib;%(AdditionalDependencies)
libprotobuf.lib;abseil_dll.lib;utf8_range.lib;utf8_validity.lib;%(AdditionalDependencies)
